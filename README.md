# Jaimon James - Personal Portfolio

This is a personal portfolio website for Jaimon James, a Data Analyst. The portfolio showcases skills, projects, and contact information.

## Features

- **Responsive Design**: The portfolio is designed to work seamlessly across all devices.
- **Modern UI**: Clean and professional user interface with a focus on readability and aesthetics.
- **Sections**:
  - **About**: Introduction and professional summary.
  - **Resume**: Detailed work experience and education.
  - **Portfolio**: Showcase of projects and achievements.
  - **Blog**: Articles and insights on data analysis and related topics.
  - **Contact**: Contact form for reaching out.

## Technologies Used

- **HTML5**: For structuring the content.
- **CSS3**: For styling and layout.
- **JavaScript**: For interactive elements and form handling.
- **Ionicons**: For icons used throughout the portfolio.

## Setup and Usage

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/Jaimon-James/Jaimon-James-Portfolio.git
   ```

2. **Navigate to the Project Directory**:
   ```bash
   cd Jaimon-James-Portfolio
   ```

3. **Open the Portfolio**:
   - Simply open the `index.html` file in your preferred web browser.

## Contact Form

The contact form is integrated with [Formspree](https://formspree.io/) for handling form submissions. Ensure you have a valid Formspree endpoint configured in the form's `action` attribute.
